    Critérios de priorização

    Nossos critérios vão de 1 a 5 e seu valor "total" é calculado pela média dos outros 3 critérios (valor de negócio , viabilidade e criticidade)

    Critérios Definição
    Valor de negócio Quanto esse requisito é importante para nossa cliente
    Viabilidade Quanto esse requisito não depende de outros + o conhecimento que nossa equipe tem para realiza-lo
    Criticidade Se esse requisito é urgente para nossa cliente

    Requisitos não-funcionais

    Categoria Requisito
    Interface [RNF01] O web site deverá ter cores e fontes padronizados.
    Interface [RNF02] O web site deverá ser responsivo.
    Suportabilidade [RNF03] O web site deverá ser compatível com as versões mais recentes dos navegadores : Mozilla , Safari , Edge , Chrome e Opera.
    Usabilidade [RNF04] O web site deverá ter modo claro e escuro.
